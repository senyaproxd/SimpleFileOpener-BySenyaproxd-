# SimpleFileOpener-BySenyaproxd-
Simple file opener that can open files by using their repository.
HOW TO USE:
1.Download the SimpleFileOpener-BySenyaproxd.py project
2.Make sure you have python installed.
3.Open the file and type file repository.(EXAMPLE C:|Users|[Your Name) until the file name.
4.MAKE SURE THAT FILE HAS " at the beginning and on the end of file repository.
5.Press enter and IF the file not found make sure that you correctly wrote the file repository.
